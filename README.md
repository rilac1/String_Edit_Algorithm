# String Edit Algorithm
- 두 개의 문자열을 입력 받아 최적의 문자열 편집 순서를 결정하는 프로그램을 작성하라.  
편집 순서가 올바른가를 테스트하기 위해 문자열 X에 적용하여 최종 결과가 문자열 Y가 되는지 확인하는 함수도 포함하라.

---
- 입력:  
두 문자열의 길이를 입력: ``` n m ```  
두 문자열을 입력:  
    ```
    x1 x2 ... xn
    y1 y2 ... ym
    ```
- 출력:  
c(i, j) 표  
최적의 편집 순서열: 맨 앞 연산부터 차례로..
입력 문자열에 차례로 적용하기

---
- 5개 이상의 문자열에 대해 적용하라.
